# `.mjs` 与 `.cjs`

- mjs 引用 cjs

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);

const commonjs = require('./commonjs.cjs');
```

- cjs 引用 mjs

```js
import('./esmodule.mjs').then((esm) => {
  // something...
});
```

- package.json

  * type="module"
  * type="commonjs"
